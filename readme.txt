== Description ==

This plugin allows admin to deactivate the widget boxes from dashboard to other user roles.  It also activates two custom boxes with WYSWYG editor that can be filled with any information.  

This plugin does the following:

- Hides default WP admin dashboards
- Creates 2 custom dashboards

== Installation ==

1. Upload the plugin PHP folder and file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin on the 'settings' menu inside 'grab the feed' submenu.
4. No other configuration needed
